#  Weather App (Python + Tkinter + OpenWeather API)

A simple and elegant **Weather Application** built using **Python**, **Tkinter**, and **Matplotlib**.  
The app allows users to check real-time weather information of any city and visualize the **10 coldest cities in Pakistan** using a bar chart.

---

##  Description

The **Weather App** is a Python-based desktop application that provides real-time weather updates using the **OpenWeatherMap API**.  
Built with **Tkinter** for the graphical interface and **Matplotlib** for data visualization, the app allows users to:

- Instantly check temperature, humidity, pressure, and weather conditions of any city.  
- View a bar chart comparing the **10 coldest cities in Pakistan**, updated dynamically.  

With its clean mobile-like interface, this project is perfect for beginners learning **API integration**, **GUI development**, and **data visualization** in Python.

---

##  Features

 Fetch real-time weather data for any city  
 Display temperature, humidity, pressure, and weather description  
 Visualize **10 Coldest Cities in Pakistan**  
 User-friendly and mobile-like GUI  
 Live data fetched from the **OpenWeatherMap API**
